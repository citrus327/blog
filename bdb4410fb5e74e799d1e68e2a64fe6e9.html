Rust - 变量和逻辑控制

变量

  • 只用 Let 申明的变量不可修改，添加 mut 变成可修改的变量，使用 mut 关键字代表在未来某个时刻变量会被变更。
  • const 申明的变量不可变，代表此变量永远不可能变更。
  • shadowing 可以重复申明变量

Data Types

  • let guess: u32 = "42".parse().expect("Not a number!");
  • :u32为类型标注，type anotation。Rust 作为静态强类型语言，某些 api 需要主动标注来进行类型判断。所以如果不写，compile 时会报错
  • Rust has four primary scalar types: integers, floating-point numbers, Booleans, and characters

Flow Control

if

  • 跟其他语言一样的 if else 逻辑控制
  • if 后的判断 expression 可以不带括号，且不会像 javascript 一样自动转成 bool
  • If 被视为一整个 expression，所以可以用类似于三元表达式的形式去表达
      • let number = if condition { 5 } else { 6 };

let x =
if 5 > 3 {
	3
} else {
	6
};
println!("The value of x is: {}", x);
      第一种写法更直接且更贴近熟悉的三元表达式。

loop

The loop keyword tells Rust to execute a block of code over and over again forever or until you explicitly tell it to stop.
fn main() {
    let mut counter = 0;

    let result = loop {
        counter += 1;

        if counter == 10 {
            break counter * 2;
        }
    };

    println!("The result is {}", result);
}
可以从 loop 中使用 break 打断 Loop，并将 break 的值返回给 counter

while

When the condition ceases to be true, the program calls break, stopping the loop. This loop type could be implemented using a combination of loop, if, else, and break;
跟一般的 while 没啥区别

for

fn main() {
    let range = 1..4; // generate a range
    let range = range.rev(); // reverse the range
    for number in range { // count from 3 to 1
        println!("{}!", number);
    }
    println!("LIFTOFF!!!");
   // 3
   // 2
   // 1
}
例子中没有提到使用 for 去完成一些 condition 判断的流程，而是直说了可以用 for 循环去遍历一些 array 或者 range。
