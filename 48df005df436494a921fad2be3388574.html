声明 Struct 结构
struct User {
    username: String,
    email: String,
    sign_in_count: u64,
    active: bool,
}
初始化 User 实例
let user1 = User {
	email: String::from("someone@example.com"),
	username: String::from("someusername123"),
	active: true,
	sign_in_count: 1,
};
读取与修改 User 实例或属性
let mut u = User {
  email: String::from("someone@example.com"),
  username: String::from("someusername123"),
  active: true,
  sign_in_count: 1,
};
u.sign_in_count = 5;

println!("{}", u.sign_in_count);
如果想要修改实例的属性，整个实例必须被标注为 mut
#[derive(Debug)]
struct Rectangle {
    width: u32,
    height: u32,
}

impl Rectangle {
    fn area(&self) -> u32 {
        self.width * self.height
    }
    fn change_width (&mut self, new_width: u32) {
        self.width = new_width;
    }
    fn create () -> Rectangle {
        Rectangle {
            width: 23,
            height: 32
        }
    }
}

fn main() {
    let mut rect1 = Rectangle::create();
    println!("rect1 is {:#?}", rect1);
    println!("rect1 area is {}", rect1.area());
    rect1.change_width(32);
    println!("rect1 area after change is {}", rect1.area());
}
为 struct 添加方法，有提到 static fn，例如 create。
