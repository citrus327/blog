React - 起步

近期后端 REST API 基本重构完成，准备基于此，会将原有 Dashboard 使用 React.js 重写。 在此过程中会陆续对使用到的技术进行归纳和总结

项目搭建

  1. 采用 Create-React-App 脚手架搭建项目
  1. 使用传统 JavaScript 而非 TypeScript，因为了解不多，防止采坑，先使用 JavaScript，会考虑后期重构
  1. 采用 Material UI 作为基础样式库/组件库。选型原因：
      • 作为基础 Dashboard 项目, 更期望采用 Material Design 来完成，因为足够简单清爽
      • Github star 数量多，老牌 React.js 维护性和口碑有保证
      • 文档足够清晰，Google Material Design Icon 分包，styles 分包
      • 样式拓展有保证

初次尝试

想要关注的问题有
  1. font-family 和字体文件如何处理
  1. 如何自定义 theme，如何书写 css 样式。目前考虑 styled-component
  1. 项目结构
  1. 底层 ISDK 封装是否要另起一个 repo 做，因为考虑会有博客端对外开放，很多 API 是共用的
  1. 不使用 hooks 开发，体验 React.js 在有 hook 之前的开发体验，能够实在的了解痛点。
  1. 状态管理选型，目前了解到的有：Redux, dva, saga 等
  1. 路由控制逻辑，中心化/去中心化？ React-Router 新版的最佳实践。
